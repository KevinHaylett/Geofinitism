# Russell's Paradox (1901)

The paradox arises in naive set theory from unrestricted comprehension: consider the set R = {x | x ∉ x} (the set
of all sets that do not contain themselves). Then:
If R ∈ R, then by definition R ∉ R → contradiction.
If R ∉ R, then by definition R ∈ R → contradiction.

This showed that Frege's system (and any similar unrestricted set theory) was inconsistent.
**How it was "handled" (not resolved away):**
The standard modern solution is **axiomatic set theory** , especially **ZFC** (Zermelo-Fraenkel set theory with the
Axiom of Choice), which restricts comprehension via the **Axiom of Separation** (or Restricted Comprehension). You
can only form {x ∈ A | φ(x)} where A is an already-existing set. This blocks the unrestricted formation of R — there
is no universal set that could serve as the domain to define it.
Alternative foundations include:
**Russell's own ramified type theory** (from _Principia Mathematica_ , with Whitehead): imposes a hierarchy of
types to prevent self-reference. It works but is cumbersome and rarely used today for mainstream math.


```
Newer type theories (e.g., in proof assistants like Coq, Agda, Lean) use simpler typed systems that avoid the
paradox.
Occasional papers propose novel "resolutions" (e.g., boundary laws, collapse theories, finitistic approaches, or
redefinitions of sets), but these are fringe, non-mainstream, and have not displaced ZFC or type theory. The
overwhelming consensus in mathematics is that ZFC (or variants like ZF + large cardinals) adequately avoids the
paradox while supporting virtually all of classical mathematics.
```
**Status in 2026:** Firmly handled/avoided in accepted foundations. Not resolved in the sense that the
unrestricted version is consistent — it isn't. The paradox demonstrates a real limitation of naive
comprehension, and modern set theory deliberately weakens that principle to restore consistency.

# Gödel's Incompleteness Theorems (1931)

These are two separate but related theorems about any formal system that is:
Consistent,
Sufficiently powerful to express basic arithmetic (Peano arithmetic or similar),
Effectively axiomatizable (axioms are recursively enumerable).

**First Incompleteness Theorem:** Such a system is incomplete — there exist true statements (in the standard
model of arithmetic) that cannot be proved or disproved within the system. (Gödel constructs a self-referential
sentence G roughly saying "G is not provable in the system" via arithmetization/Gödel numbering.)
**Second Incompleteness Theorem:** If the system is consistent, it cannot prove its own consistency.
**Have they been resolved or bypassed?**
No mainstream development has overturned them. They are theorems — proven rigorously — and apply to
essentially all interesting formal systems of mathematics.
There are weak systems that escape the second theorem (e.g., some sub-systems of arithmetic without full
multiplication that can prove their own consistency), but these are deliberately too weak for most mathematics.
Rosser's strengthening (1936) removed reliance on ω-consistency, making the first theorem even more robust.
Computer-verified proofs (e.g., in Coq, Isabelle, HOL Light) have mechanically confirmed the theorems in recent
decades, reinforcing their solidity.
Philosophical interpretations vary (e.g., implications for mind vs. machine, AI limits, Hilbert's program being
impossible), but the mathematical statements stand unchallenged.
Fringe or speculative claims occasionally appear (e.g., "resolving" via resonance, structured hierarchies, or
reinterpretations), but these are not accepted in logic/mathematics communities and do not invalidate the original
proofs.

**Status in 2026:** Still fully valid and central. They destroyed Hilbert's dream of a complete, consistent, finitary
foundation for all mathematics. Modern mathematics lives happily with incompleteness: we accept that some
truths (like the Continuum Hypothesis in ZFC) are independent, and we add axioms as needed for specific
domains.

# How medical device regulation handles precision claims

FDA (and equivalent bodies like EMA, MHRA) do **not** require "5σ discovery-level" statistics for most approvals,
especially not for high-risk (Class III) life-sustaining devices. Instead, they emphasize:
**Reliability targets** — Often framed as "95% confidence in 95% reliability" (R95/C95) for critical failure modes.

rates, not chasing tail probabilities in the 10⁻⁷ range.
**Clinical performance** — For diagnostics or monitoring devices, sensitivity/specificity with **95% confidence

appears in some pivotal studies, but it's not a universal gate. Equivalence/non-inferiority trials are common for
510(k) clearances, where showing "no clinically meaningful difference" trumps proving superiority with extreme
confidence.
**Post-market surveillance** — Mandatory reporting of adverse events, recalls if patterns emerge. Many devices get
